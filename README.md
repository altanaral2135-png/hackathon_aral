# hackathon_aral

## EkoTekstil – Sürdürülebilir Tekstil Geri Dönüşüm Web Sitesi

## Proje Amacı
Bu proje, tekstil geri dönüşümü alanında faaliyet gösteren kurgusal bir firma için hazırlanmış
modern ve etkileşimli bir tanıtım web sitesidir.

Amacım:
- HTML, CSS ve JavaScript bilgimi gerçekçi bir senaryo üzerinde uygulamak
- Animasyonlar ve kullanıcı etkileşimleriyle modern frontend mantığını öğrenmek
- GitHub profilimde tamamlanmış ve açıklanmış bir proje sunmak

---

## Proje Özellikleri
- Tek sayfa (SPA benzeri) yapı
- Smooth scroll navigasyon
- CSS ile 3D animasyonlar
- Responsive tasarım
- FAQ açılır-kapanır yapı (JavaScript)
- Sürdürülebilirlik temalı modern kullanıcı arayüzü

---

## Kullanılan Teknolojiler
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Inter)
